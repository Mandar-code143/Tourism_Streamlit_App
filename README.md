Project Overview

The Tourism Experience Analytics System is an end-to-end Machine Learning project designed to analyze tourism-related data and predict user travel behavior.

This project performs:

Tourism data preprocessing
Exploratory Data Analysis (EDA)
Machine Learning model training
Visit mode prediction
Attraction rating prediction
Recommendation system implementation
Streamlit cloud deployment

The system helps analyze tourism patterns and generate intelligent predictions using Machine Learning algorithms.

Features

✅ Tourism data preprocessing and cleaning
✅ Multiple dataset merging using Pandas
✅ Exploratory Data Analysis (EDA)
✅ Classification model for Visit Mode Prediction
✅ Regression model for Rating Prediction
✅ Recommendation System using Cosine Similarity
✅ Interactive Streamlit Web Application
✅ Cloud Deployment using Streamlit Cloud
